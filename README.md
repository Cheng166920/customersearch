# 客户资料地图查询系统

## 技术栈
```
Vue CLI  + Vue2  +  Axios  +  ElementUI  +  JavaScript API GL(百度地图)  +  JavaScript

```

### 项目描述
```
Web 端全国客户资料地图查询系统，实现属性检索、要素查询、热力图显示等功能。
```

### 技术说明
```
• 实现虚拟列表，并通过节流优化滚动，优化查询性能。
• 使用百度地图 API 提供的要素聚合功能，提高渲染效率。
```
